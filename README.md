# Cipher Rogue 

## Requirement
### Arch Linux
1. C++ compiler:

    ```bash
    sudo pacman -S gcc
    ```

2. SFML Library:

    ```bash
    sudo pacman -S sfml
    ```

### Ubuntu
1. C++ compiler:

    ```bash
    sudo apt update
    sudo apt install g++
    ```

2. SFML Library:

    ```bash
    sudo apt update
    sudo apt install libsfml-dev
    ```

### Fedora
1. C++ compiler:

    ```bash
    sudo dnf install gcc-c++
    ```

2. SFML Library:

    ```bash
    sudo dnf install SFML-devel
    ```

## Building

1. Build the project using the makefile:
    
    ```bash
    make
    ```

2. Run the main file:

    ```bash
    ./bin/main
    ```
